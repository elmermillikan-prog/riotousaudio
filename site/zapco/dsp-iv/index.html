Zapco

DSP-IV Series

The DSP-IV processors and amplified DSP — Zapco's modern digital signal-processing core.

The Zapco DSP-IV Series brings audiophile-grade digital signal processing to an accessible price point — a lineage that traces back to the original DSP-Z8 IV, which Zapco credits with performance superior to competing units costing twice as much and even earning use in top-tier Master Class SQ competition vehicles. The current generation, the DSP-IV AT, refines that legacy with a sharp new black-and-copper aesthetic matching the AP amplifier family, adding coaxial digital input alongside optical, and introducing the AutoTune auto-calibration function for AT-designated models.

The series spans a standalone 8-channel processor (DSP-Z8 IV AT) and three integrated DSP/amplifier units (ADSP-Z8 IV-6AT, ADSP-Z12 IV-10A, ADSP-Z16 IV-12A). All models share the CS47048 32-bit/192 kHz DSP engine, 31-band graphic/parametric EQ, four crossover filter families, per-channel time alignment up to 515 cm, and a port for the optional HD-BT II-D module providing aptX HD Bluetooth streaming. The GUI's Advanced menu adds a Mixing Set for manual input-to-channel routing and, on AT units, the AutoTune function that automatically sets signal delay and channel phase using an optional calibration microphone.
